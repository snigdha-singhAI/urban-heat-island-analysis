# Urban Heat Island (UHI) Analysis Using Data Analytics and Machine Learning

## Project Status

🚧 Ongoing Research Project

This project is currently under development as part of an ongoing research initiative focused on environmental analysis and urban sustainability.

## Overview

This project aims to analyze the Urban Heat Island (UHI) effect using data analytics, visualization techniques, and machine learning approaches.

The objective is to identify temperature variation patterns, environmental factors contributing to urban heat accumulation, and potential data-driven insights that can support sustainable urban planning.

## Current Work

The project currently includes:

* Data collection and preprocessing
* Exploratory Data Analysis (EDA)
* Environmental data visualization
* Feature analysis
* Initial machine learning experimentation
